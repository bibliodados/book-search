# 📚 Book Search App

Uma aplicação web robusta concebida e desenvolvida por **Marcelo Diniz** para buscar metadados de livros em múltiplas bases globais e formatá-los de acordo com padrões rigorosos (ABNT NBR 6023, MARC 21, e Dublin Core).

## 🚀 Funcionalidades

- **Busca Unificada:** Pesquise por título, autor, assunto ou ISBN.
- **Integração com Múltiplas Bases:** OpenLibrary, Google Books, Internet Archive, Gutendex (Projeto Gutenberg), Wikisource e Wikidata.
- **Formatador Automático:**
  - Referências bibliográficas completas no padrão *ABNT NBR 6023:2018*.
  - Códigos catalográficos em *MARC 21*.
  - Metadados semânticos em *Dublin Core*.
- **Download de Dados:** Exporte facilmente o resultado consolidado em formato `.json`.
- **Tema Premium:** Interface minimalista, veloz e com suporte a *Tema Escuro*.
- **Single-File Ecosystem:** Todo o código — HTML, CSS estiloso e todo o motor assíncrono em Javascript — habita um mesmo e levíssimo arquivo HTML.

## 🛠️ Tecnologias Utilizadas

- **HTML5 & CSS3** (Variáveis CSS, CSS Grid/Flexbox)
- **Vanilla JavaScript** (Sem nenhum framework, máxima performance)
- **APIs REST** públicas para a coleta dos metadados.
